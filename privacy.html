Privacy Policy

Last updated: 2025-11-02

1. Summary

This policy describes what data “Ref Bot by Star” processes, why, and how to contact us.

2. Data We Process

  • Discord IDs and metadata: user ID, guild ID, channel ID, message ID, invite code, and usernames/tags as provided by Discord.
  • Referral events: who invited whom, when a member joined, and invite use counts.
  • Validation actions: first message in configured channels; reactions to a checkpoint message.
  • Technical logs and error data for reliability and abuse prevention.

3. Purpose

  • Operate core features (personal invite links, referral counts, leaderboards, breakdowns).
  • Prevent abuse and maintain security.
  • Comply with legal requests when required.

4. Storage & Retention

Referral/action data is stored in a local database controlled by the server operator hosting the bot. Data is retained as long as necessary for features or until deletion is requested by a server admin.

5. Sharing

No sale of personal data. Data may be shared with Discord as part of normal API operations or with authorities if legally required.

6. Your Choices

  • Users can leave a server to stop further processing.
  • Server admins can remove the bot to stop processing for that server.
  • Data deletion/export requests can be sent to the contact below; server admins can request purge of their server’s stored data.

7. Security

We use reasonable technical and organizational measures, including restricted token access and least-privilege permissions.

8. Children

The Service does not target children; use must comply with Discord’s age requirements.

9. International

Processing occurs where the bot is hosted. Your data may be transferred according to hosting location.

10. Changes

We may update this policy; continued use after updates constitutes acceptance.
